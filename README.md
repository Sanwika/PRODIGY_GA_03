# PRODIGY_GA_03 - Text Generation with Markov Chains

## Objective
Build a text generation model using Markov Chains to generate coherent and contextually relevant text based on the patterns learned from a given dataset.

## Description
This project implements a simple Markov Chain text generator using Python. The model analyzes relationships between words in the input text and generates new text by predicting the next word based on the current word.

## Technologies Used
- Python
- Google Colab

## Features
- Learns word sequences from input text
- Generates new text based on learned patterns
- Demonstrates probabilistic text generation

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells.
3. View the generated text output.
